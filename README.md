🍅 ZenTimer - 极简番茄钟

一个简洁优雅的番茄钟（Pomodoro Timer）网页应用，帮助你保持专注，提升工作效率。

![ZenTimer](screenshot.png)

## ✨ 特性

- **三种模式**：专注（25分钟）、短休息（5分钟）、长休息（15分钟）
- **自定义设置**：可自由调整各模式的时长
- **桌面通知**：计时结束时弹出浏览器通知
- **声音提示**：使用 Web Audio API 生成提示音，无需音频文件
- **进度追踪**：自动记录完成的番茄数量
- **数据持久化**：使用本地存储保存设置和进度
- **响应式设计**：完美适配桌面端和移动端
- **暗色主题**：护眼舒适的深色界面

## 🚀 使用

### 直接使用

直接在浏览器中打开 `index.html` 即可使用。

### 本地服务器

推荐使用本地服务器以获得最佳体验：

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js http-server
npx http-server
```

然后访问 `http://localhost:8000`

## 💡 技术栈

- 纯原生 JavaScript（ES6+）
- HTML5
- CSS3（CSS 变量、Flexbox）
- 无任何框架依赖

## 📸 界面预览

- **专注模式**：红色主题，帮助你集中注意力
- **短休息**：绿色主题，让眼睛放松
- **长休息**：蓝色主题，深度休息

## 🎯 使用建议

1. 设置 25 分钟专注时间
2. 全身心投入工作，避免分心
3. 计时结束后休息 5 分钟
4. 每完成 4 个番茄，进行一次 15 分钟的长休息
